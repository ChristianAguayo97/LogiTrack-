# LogiTrack

Sistema de gestión de envíos con FastAPI y Python.

## Estructura

```
/docs       - Documentación
/src        - Código fuente
/tests      - Pruebas unitarias
```

## Instalación

```bash
pip install -r requirements.txt
```

## Ejecución

```bash
cd src
uvicorn main:app --reload
```

## Tests

```bash
pytest tests/
```
